# Guía de uso para la cliente

## 1. Objetivo de esta guía

Esta guía explica de forma simple cómo usar el sitio **My Light Gallery Art**, cómo operar el catálogo y cómo aprovechar las herramientas que ya están activas.

## 2. Qué puede hacer hoy en el sitio

Actualmente la web permite:

- mostrar la galería en español e inglés;
- presentar la biografía de la artista;
- exhibir obras disponibles;
- abrir una ficha individual por obra;
- recibir mensajes desde el formulario de contacto;
- recibir registros desde el formulario de newsletter;
- compartir obras en redes;
- generar un certificado de autenticidad para una obra.

## 3. Navegación básica

### Página principal

En la página principal el visitante puede:

- conocer la galería;
- ver obras destacadas;
- entrar al catálogo completo;
- leer información de Layla;
- cambiar de idioma.

### Catálogo de obras

En la sección de exhibiciones el visitante puede:

- ver varias obras;
- pasar entre páginas del catálogo;
- entrar al detalle de cada pieza.

### Ficha individual de obra

Dentro de cada obra el visitante puede:

- ver la imagen principal;
- consultar técnica y dimensiones;
- compartir la pieza;
- contactar a la galería.

## 4. Cómo actualizar obras

### Importante

El catálogo no se edita dentro de este sitio. La información proviene del sistema **WordPress/WooCommerce** conectado a la web.

Si usted ya tiene acceso a ese panel, puede actualizar las obras desde ahí. Si todavía no tiene acceso, debe solicitarlo a **O7 Digital**.

### Datos que la web utiliza de cada obra

Para que una pieza se muestre correctamente, conviene completar al menos:

- nombre de la obra;
- slug;
- imagen principal;
- descripción;
- técnica;
- dimensiones;
- precio, si desea mostrar referencia comercial.

### Recomendaciones de carga

- Usar una imagen principal clara y de buena calidad.
- Escribir el nombre de la obra de manera consistente.
- Mantener técnica y dimensiones bien capturadas.
- Revisar que la descripción no quede vacía.
- Publicar la obra una vez validada.

### Después de publicar

Una vez guardada o publicada la obra en WordPress/WooCommerce, el sitio la toma desde la API. Si el cambio no aparece de inmediato, puede tardar unos minutos por caché.

## 5. Formularios del sitio

### Formulario de contacto

Sirve para que visitantes o compradores potenciales envíen:

- nombre;
- correo;
- teléfono;
- asunto;
- mensaje.

### Formulario de newsletter

Sirve para captar suscriptores interesados en:

- novedades;
- exposiciones;
- promociones;
- nuevas obras.

### Recomendación operativa

- Revisar con frecuencia el correo receptor configurado.
- Responder rápido a consultas sobre disponibilidad y precio.
- Revisar también la carpeta de spam o promociones.

Si desea cambiar el correo que recibe estos formularios, ese ajuste debe gestionarse con **O7 Digital**.

## 6. Cómo generar el certificado de autenticidad

La ficha de cada obra incluye la opción **“Generar certificado COA”**.

### Pasos

1. Entrar a la obra correspondiente.
2. Hacer clic en **“Generar certificado COA”**.
3. Ingresar el usuario y la contraseña entregados por O7 Digital.
4. Completar el nombre del cliente.
5. Confirmar la fecha de emisión.
6. Hacer clic en **“Descargar certificado en PDF”**.

### Importante

- El acceso al COA es restringido.
- La sesión dura varias horas una vez validada.
- El PDF se genera desde el navegador.
- Si cambia el navegador o se cierra la sesión, puede ser necesario volver a autenticarse.

## 7. Idiomas

El sitio tiene versión en:

- español;
- inglés.

El visitante puede cambiar el idioma desde el encabezado. La estructura principal del sitio ya está duplicada para ambos idiomas.

## 8. Lo que no hace todavía el sitio

Para evitar confusiones, conviene tener presente que hoy la web:

- no tiene un carrito de compra activo;
- no tiene checkout de pago implementado en este repositorio;
- no tiene panel administrativo propio dentro del sitio;
- no tiene CRM integrado todavía.

Hoy la web funciona principalmente como:

- presentación profesional;
- catálogo de obras;
- captación de leads;
- herramienta comercial de apoyo.

## 9. Buenas prácticas para operar la web

- Mantener el catálogo actualizado.
- Responder consultas en menos de 24 horas.
- Verificar que las imágenes nuevas se vean bien.
- Revisar periódicamente enlaces, textos y datos de contacto.
- Pedir apoyo técnico a O7 Digital cuando se necesiten cambios estructurales.

## 10. Soporte recomendado

Para cambios como los siguientes, se recomienda solicitar apoyo a O7 Digital:

- cambios de diseño;
- cambios de textos legales;
- nuevas secciones;
- conexión con CRM;
- automatizaciones comerciales;
- mejora del chat;
- cambios de dominio, hosting o analítica.
